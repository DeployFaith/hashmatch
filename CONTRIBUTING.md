# Contributing

Thanks for contributing to HashMatch.

## Development setup

```bash
npm install
```

## Quality checks

```bash
npm run lint
npm run format:check
npm run typecheck
npm test
```

## Branching and PRs

- Create a feature branch off `main`.
- Keep pull requests focused and scoped.
- Include a brief summary and testing notes in the PR description.
- Ensure all checks pass before requesting review.
